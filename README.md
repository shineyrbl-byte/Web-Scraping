# Remote Job Market Intelligence - Ethical Web Scraping Project
## Project Overview
This project scrapes job listings from Remote OK (remoteok.com)
to analyze market trends in remote employment. The analysis
covers skill demand, job type distribution, and geographic patterns.
## Data Source
- Website: https://remoteok.com
- Compliance: Adheres to robots.txt (1-second crawl delay)
- Scraping Date: 28-02-2026
